# Road Accident Hotspot Detection using Machine Learning

This project analyzes road accident data to detect accident hotspots using clustering and geospatial analysis.

## Technologies
- Python
- Pandas
- Scikit-learn
- Folium
- Matplotlib

## Methods
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- DBSCAN Clustering
- Geospatial Visualization

## Output
Interactive accident hotspot heatmap of the UK.
